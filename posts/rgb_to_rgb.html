The setup/The plan

The basic plan we will follow is this:

  1. Load Image from Disk (CPU-side)
  2. Create Staging Buffer (CPU→GPU transfer workspace)
  3. Create Image (GPU texture storage)
  4. Create ImageView (how to interpret the image)
  5. Create Sampler (how to read the texture)
  6. Transfer Image Data (CPU→GPU)
  7. Create Descriptor Set Layout
  8. Update Graphics Pipeline
  9. Create Descriptor Pool
  10. Allocate Descriptor Set
  11. Update Descriptor Set (bind resources)
  12. Begin Command Buffer
  13. Begin Render Pass
  14. Bind Pipeline
  15. Bind Descriptor Set
  16. Bind Vertex Buffer
  17. Bind Index Buffer (if using indexed drawing)
  18. Draw
  19. End Render Pass & Command Buffer
  20. Submit & Present
  21. Cleanup (Shutdown)
